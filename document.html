معرفی مستندات

به مستندات پروژه Afghan Code خوش آمدید! این راهنما به شما کمک می‌کند تا به راحتی کد مینی‌اپ را درک کرده و آن را بر اساس نیازهای تیم خود شخصی‌سازی کنید. این پروژه با استفاده از تکنولوژی‌های مدرن و محبوب فرانت‌اند مانند TailwindCSS، GSAP و Three.js ساخته شده تا یک تجربه بصری خیره‌کننده ارائه دهد.

هدف اصلی این پروژه، ارائه یک ویترین حرفه‌ای و جذاب برای معرفی تیم توسعه، خدمات و دستاوردهای آن است. تمام کدها در یک فایل واحد (`index.html`) نوشته شده‌اند تا استفاده و ویرایش آن به ساده‌ترین شکل ممکن باشد.

ساختار کلی فایل

کد پروژه به چند بخش اصلی تقسیم شده است که درک آن‌ها برای ویرایش آسان ضروری است.

  • <head>: در این بخش، تمام وابستگی‌ها مانند TailwindCSS، فونت وزیر، GSAP، Three.js و Prism.js (برای هایلایت کد) از طریق CDN فراخوانی شده‌اند. همچنین، استایل‌های سفارشی در تگ <style> قرار دارند.
  • <body>: بدنه اصلی صفحه که شامل تمام بخش‌های قابل مشاهده است.
  • <header>: هدر هوشمند صفحه که در اسکرول ظاهر می‌شود.
  • <main>: محتوای اصلی صفحه که به چند <section> تقسیم شده است: بخش Hero، خدمات، تیم، انجمن‌ها و بلاگ.
  • <footer>: فوتر ساده و زیبای پروژه.
  • <script>: در انتهای فایل، تمام منطق جاوااسکریپت برای داینامیک کردن محتوا، مدیریت انیمیشن‌ها و تعاملات کاربر قرار دارد.

شخصی‌سازی داده‌ها

یکی از مهم‌ترین بخش‌های این پروژه، قابلیت شخصی‌سازی آسان محتوا از طریق ویرایش متغیرهای جاوااسکریپت است. تمام داده‌های متنی و تصویری در ابتدای تگ <script> در قالب آرایه‌هایی از اشیاء (Arrays of Objects) تعریف شده‌اند.

افزودن یک عضو جدید به تیم

برای اضافه کردن یک عضو جدید به تیم، کافیست یک آبجکت جدید به آرایه teamData اضافه کنید. ساختار این آبجکت باید مانند نمونه زیر باشد: 

// In the teamData array, add a new object:
{ 
    name: 'نام عضو جدید', 
    role: 'سِمَت عضو', 
    skills: ['مهارت ۱', 'مهارت ۲'], 
    avatar: 'https://placehold.co/120x120/......', // Profile image link
    socials: { github: '#', linkedin: '#' }, 
    glowColor: 'var(--glow-color-sky)' // Card glow color
}

پس از ذخیره کردن فایل، عضو جدید به صورت خودکار به لیست اعضای تیم اضافه خواهد شد و تمام انیمیشن‌ها و استایل‌ها برای آن اعمال می‌شود.

استایل و انیمیشن

تغییر استایل‌ها با TailwindCSS

این پروژه به طور کامل با TailwindCSS استایل‌دهی شده است. این به شما اجازه می‌دهد تا بدون نوشتن CSS سفارشی، به راحتی ظاهر هر المان را تغییر دهید. برای مثال، برای تغییر رنگ یک دکمه، کافیست کلاس‌های bg- و text- مربوط به آن را ویرایش کنید.

منطق انیمیشن‌ها با GSAP

تمام انیمیشن‌های این پروژه با استفاده از کتابخانه قدرتمند GSAP پیاده‌سازی شده‌اند. منطق انیمیشن‌ها در انتهای فایل جاوااسکریپت قرار دارد. 

// Example: Team card entrance animation
gsap.from(".holographic-card", { 
    scrollTrigger: { 
        trigger: "#team-section", // Activates when the team section is reached
        start: 'top 75%' // Animation starts when 75% of the section is in view
    }, 
    opacity: 0, 
    y: 100, 
    duration: 1, 
    stagger: 0.2, // Delay between each card's animation
    ease: 'power3.out' 
});

با تغییر پارامترهای این توابع، می‌توانید رفتار، سرعت و نوع انیمیشن‌ها را به طور کامل شخصی‌سازی کنید. استفاده از `ease` های مختلف مانند `elastic.out` یا `back.out` می‌تواند به انیمیشن‌های شما حس طبیعی‌تری ببخشد.
